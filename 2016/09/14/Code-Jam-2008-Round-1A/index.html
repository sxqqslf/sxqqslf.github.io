Code Jam Round 1A

Problem A. Minium Scalar Product

给定两个向量v1=(x1,x2,…xn)和v2=(y1,y2,…yn)，适当调整向量内部的顺序，使得他们的数量积最小、最大

分析

贪心，对向量升序，最小值和最大值分别对应于两者顺序相乘和逆序相乘

Problem B. Milkshakes

你有N种不同的奶昔可供用户选择，每种奶昔可以被调制成0或1，同时有M个顾客，每个人对不同的奶昔有不同的喜 好，并且最多只喜欢一种奶昔，问是否存在一种奶昔的调制方案，使得所有用户都可以被满足

分析

此题的突破点在于每个顾客最多只喜欢一种奶昔

  1. 从所有奶昔都被调制成0开始，遍历所有用户
  2. 如果一个用户的奶昔偏好都是0，但是这些奶昔目前都被调制成了1，那么不存在调制方案
  3. 如果存在一个用户仅喜欢一个奶昔，且偏好为1，那么该奶昔就必须调制成1
  4. 如果所有用户都满足，则得到一个方案

上述的关键在于，将一个奶昔调制成1后，不会再回溯将其调制成0了，因为这是强制的操作

Problem C. Numbers

计算(3+√5)n的整数部分的末3位，比如(3+√5)5=3935.73982，答案是935
